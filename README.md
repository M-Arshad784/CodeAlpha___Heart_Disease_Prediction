# CodeAlpha – Heart Disease Prediction

This project is a **Machine Learning-based Heart Disease Prediction System** developed as part of the **CodeAlpha Internship**. The model predicts whether a person is at risk of heart disease based on medical and health-related attributes.

## Features

* Data preprocessing and feature engineering
* Exploratory Data Analysis (EDA)
* Training and evaluation of multiple machine learning models
* Performance comparison using classification metrics
* Model saving for future prediction and deployment

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Dataset

The project uses a heart disease dataset containing patient health information such as age, sex, blood pressure, cholesterol level, heart rate, and other clinical features.

## Objective

The objective of this project is to build an accurate machine learning model that assists in the early prediction of heart disease, helping support timely medical decision-making.

## Results

Multiple machine learning algorithms were trained and evaluated, and the best-performing model was selected based on accuracy and other classification metrics.
